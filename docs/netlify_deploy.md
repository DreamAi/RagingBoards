RAGING BOARDS
Netlify Deployment

STEP 1
Create account

https://netlify.com

STEP 2
Install Netlify CLI

npm install -g netlify-cli

STEP 3
Login

netlify login

STEP 4
Build frontend

cd frontend

npm run build

STEP 5
Deploy

netlify deploy --prod

STEP 6
Netlify gives you live URL

Example

https://ragingboards.netlify.app
